Welcome to Package Registration Service

Registration Steps!
  • 1. Place the Package on Weight Sensor.
  • 2. Select "Send Package".
  • 3. Enter Required Information As Mentioned.
  • 4. Make the Payment Using the ATM Machine.
  • 5. Paste the Barcode in the Receipt on the Package..
  • 6. Place the Package on Weight Sensor and Scan the Barcode Using Barcode Reader to Confirmation.
